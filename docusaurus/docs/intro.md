# Homelab

Een 3-node **Proxmox-cluster** met **hoog-beschikbare Kubernetes** op VM's, volledig als
code beheerd (Terraform + Ansible + GitOps). Deze kennisbank legt vast **waarom** het zo
gebouwd is en **hoe** je het draait.
